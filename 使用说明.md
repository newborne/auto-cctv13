# CCTV13 自动播放监控脚本 - 中文说明

## 快速开始

### 1. 安装 Python

访问 https://www.python.org/downloads/ 下载 Python 3.8+
- **重要**: 安装时务必勾选 "Add Python to PATH"

### 2. 一键安装

双击运行：`install.bat`（Windows）或 `install.sh`（macOS/Linux）

### 3. 运行脚本

```bash
python cctv13_auto_play.py
```

### 4. 停止脚本

按 `Ctrl+C`

## 注意事项

1. **首次运行慢**：需下载浏览器内核，1-5 分钟
2. **全屏问题**：如不全屏，手动双击视频一次
3. **杀毒软件**：可能被拦截，添加例外
4. **日志文件**：`cctv13_auto_play.log`

## 常见问题

### Python 未安装？
安装 Python 3.8+，勾选 PATH

### F11 不生效？
手动双击视频全屏即可

### 视频暂停？
脚本会自动恢复，无需担心

### 首次运行慢？
耐心等待浏览器下载完成
